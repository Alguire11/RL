UAT Test Script: Tenant Module (MVP)
Purpose

To verify that tenants can register, log in, link rent payments, track credit progress, and export verified rent ledgers, all within MVP scope.

1. Tenant MVP Feature Scope
Feature	Description	Included in MVP
User Registration/Login	Tenant creates account via invite or open registration	✅
Link Property / Accept Invite	Tenant links to a landlord via email token or property code	✅
Record Rent Payment	Upload or log rent payment proof (bank statement, receipt, etc.)	✅
View Rent History	See list of past rent payments and their verification status	✅
Export Verified Ledger	Download verified rent history PDF	✅
Credit Progress Tracker	Basic progress bar showing “months verified”	✅
Notifications	Alerts when rent is verified or rejected by landlord	✅
Profile Settings	Edit name, address, or bank info	✅
Support Form	Contact support for help	✅
2. Functional Requirements (For UAT Testing)
ID	Requirement	Description / Expected Behaviour	MVP
FR-T1	User Registration	Tenant can create an account via email/password or via invite token.	✅
FR-T2	Login Authentication	Tenant can log in securely and remain authenticated.	✅
FR-T3	Link to Property	Tenant can join a property using code/invite link from landlord.	✅
FR-T4	Upload Rent Proof	Tenant uploads rent payment (file or input amount/date).	✅
FR-T5	View Payment History	Tenant can view all submitted payments with status.	✅
FR-T6	Notification Alerts	Tenant receives alert when landlord verifies/rejects payment.	✅
FR-T7	View Credit Progress	Progress tracker updates based on verified payments.	✅
FR-T8	Download Verified Ledger (PDF)	Tenant can download full ledger for references or credit use.	✅
FR-T9	Edit Profile	Tenant can update personal info, bank name, etc.	✅
FR-T10	Support Request	Tenant can send inquiries via a simple form.	✅
3. Non-Functional Requirements
Category	Requirement	Description
Security	End-to-end encryption for payment records.	Use HTTPS & AES-256 encryption.
Performance	Dashboard loads in under 3 seconds.	Required.
Usability	Mobile-first responsive design.	Required (tenant app = mobile-first).
Compliance	GDPR and ICO-compliant data collection.	Required.
Reliability	Rent records persist after logout.	Required.
Audit Logging	All uploads timestamped and linked to user ID.	Required.
